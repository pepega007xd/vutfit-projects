The application itself is implemented in the app/ folder.
The following files and folders are located inside app/.

-- frontend (xbrabl04) --

page.tsx - Home page
new_post/ - Page for creating a post
post/ - Components for displaying and editing posts, and for sorting and filtering

-- frontend (xhashm00) --

navigation.tsx - navigation panel
galleries/ - Components for galleries and subgalleries
public/ - Page for public galleries
user/ - Page for user galleries

-- backend/other --

actions.ts - backend implementation
theme.tsx - Theme setting for MaterialUI
layout.tsx - Basic page layout
image/ - API endpoint for returning images
